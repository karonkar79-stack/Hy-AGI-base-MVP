🚀 Quick Start

1. Clone Repository

git clone https://github.com/karonkar79-stack/Hy-AGI-base-MVP.git

2. Install & Run

cd Hy-AGI-base-MVP/backend
npm install
docker-compose up -d
npm run dev

3. Test API

curl http://localhost:3000/api/health

📡 API Endpoints

GET /api/health Health check
GET /api/agents List available agents
POST /api/tasks Execute agent task
GET /api/metrics System metrics
GET /api/demo/findings Demo security findings
